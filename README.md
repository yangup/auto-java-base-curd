# auto-java-base

自动生成一些基础CURD的代码

#### 下载项目

``` bash
导入到 eclipse 或者 idea

配置依赖 
WebContent/WEB-INF/lib

然后
到src/u.auto/main/AutoMain配置一些数据库参数

最后
运行
src/u.auto/main/AutoMain
就可以了

项目中的 auto.demo 我生成的一个 demo

诸位觉得有用,就可以下载下来,用用,代码,你可以自己修改一下,让生成的模板,更适应你现在的项目,
很佛系

```
